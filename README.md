# React 课程管理系统

基于 React 构建的课程管理实验项目，实现了完整的课程管理功能，包括课程的增删改查、状态管理、搜索筛选等。

## 🚀 项目功能

### 基础功能
- ✅ 显示课程列表
- ✅ 展示课程名称与课程简介
- ✅ 新增课程（包含输入验证）
- ✅ 删除课程
- ✅ 学习按钮交互（切换学习状态）
- ✅ 列表渲染
- ✅ 状态管理
- ✅ 组件间数据传递
- ✅ 输入校验（课程名称和简介不能为空）
- ✅ 基础样式效果

### 挑战功能
- ✅ 课程搜索功能
- ✅ 课程分类筛选功能
- ✅ 课程数量统计
- ✅ 编辑课程功能
- ✅ 优化页面样式与交互效果

## 📁 项目结构

```
src/
├── components/
│   ├── Header.jsx          # 页面头部组件
│   ├── CourseList.jsx      # 课程列表组件
│   ├── CourseCard.jsx      # 课程卡片组件
│   └── Footer.jsx          # 页面底部组件
├── App.jsx                 # 主应用组件
├── App.css                 # 全局样式文件
└── main.jsx               # 应用入口文件
```

## 🛠 技术栈

- **React 18** - 函数组件
- **JSX** - 语法
- **useState** - 状态管理
- **Props** - 组件传值
- **CSS3** - 样式设计
- **Vite** - 构建工具

## 📋 核心组件说明

### App.jsx
主应用组件，包含：
- 课程数据状态管理
- 新增课程功能
- 删除课程功能
- 编辑课程功能
- 搜索和筛选功能
- 课程统计功能

### Header.jsx
页面头部组件，展示：
- 系统标题
- 功能描述

### CourseList.jsx
课程列表组件，负责：
- 渲染课程列表
- 处理空状态显示
- 传递课程数据给 CourseCard

### CourseCard.jsx
课程卡片组件，包含：
- 课程信息展示
- 学习状态切换
- 编辑课程功能
- 删除课程功能
- 分类标签显示

### Footer.jsx
页面底部组件，展示：
- 版权信息
- 功能特色说明
